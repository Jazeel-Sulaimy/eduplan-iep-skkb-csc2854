IEP SKKB - RUN INSTRUCTIONS (WINDOWS / XAMPP)
================================================

1. Extract this ZIP into a folder, for example:
   C:\xampp\htdocs\IEP_SKKB_FIXED

2. Open PowerShell inside that exact folder.
   Confirm that BOTH files are visible:
   Get-ChildItem artisan
   Get-ChildItem composer.json

3. Start Apache and MySQL in XAMPP.

4. Create an empty MySQL database named:
   iep_skkb

5. Confirm the .env database settings:
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=iep_skkb
   DB_USERNAME=root
   DB_PASSWORD=

6. Install dependencies only if the vendor folder is missing:
   composer install

7. Clear cached files:
   php artisan optimize:clear

8. Apply only new migrations without deleting existing records:
   php artisan migrate

9. For a completely new/empty database, load demo data once:
   php artisan db:seed

10. Validate the project:
    php artisan route:list
    php artisan test

11. Start the system:
    php artisan serve

12. Open:
    http://127.0.0.1:8000

Demo accounts after seeding:
- ADMIN001 / 12345
- TEACHER001 / 12345
- COUNSELLOR001 / 12345
- PARENT001 / 12345
- SYSADMIN001 / 12345

Do not run php artisan migrate:fresh on a database containing important records.
